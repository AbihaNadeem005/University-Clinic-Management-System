# Clinic Management System

## Overview
This is a desktop-based Clinic Management System developed using C#. The system is designed to manage patient records, including treatment details and medicine information. It also includes a simple login authentication system.

---

## Features
- Simple login authentication (demo-based)  
- Patient record management  
- Patient treatment details management  
- Medicine record management

---

## Technologies Used
- C#  
- Windows Forms (.NET Framework)  
- SQL Server  
- Visual Studio

## How to Run the Project

1. Open the project in Visual Studio
2. Restore required dependencies (if any)
3. Configure SQL Server connection in the project
4. Build and run the project using Start (F5) 

---

## Database Setup

The system uses SQL Server for storing clinic data including patients, treatments, and medicine records.

---

## Purpose
This project was developed as a university project to automate clinic record management by replacing manual record keeping with a digital system using C#. It improves efficiency in storing and accessing patient, treatment, and medicine data.
